Add the TalkToGo live chat widget to your WordPress site and chat with your visitors in realtime.

== Description ==

TalkToGo adds a customizable live chat widget to your WordPress site:

* Chat with visitors in realtime from the TalkToGo dashboard
* Get notified the moment a new visitor lands on your site
* See each visitor's country, browser, and the page they are currently viewing — live, including page changes
* Customize the widget colors, texts and position per site from the TalkToGo dashboard

== Installation ==

1. Upload the plugin zip via Plugins → Add New → Upload Plugin, then activate it.
2. In your TalkToGo dashboard, open your website's Settings page and copy the Site ID and Widget URL.
3. In WordPress, go to Settings → TalkToGo, paste both values and save.
4. Visit your site — the chat bubble appears in the corner.

== Frequently Asked Questions ==

= Where do I get a Site ID? =

Create a free account in your TalkToGo dashboard, add your website, and copy the Site ID from the Settings page.

= Does it slow down my site? =

No. The widget script loads asynchronously and weighs well under 100 KB gzipped.
